# TradeSphere 📈

A full-stack stock trading platform inspired by Zerodha, built from scratch using the MERN stack.

## 🚀 Live Demo
Coming soon...

## 🛠️ Tech Stack
- **Frontend:** React.js, Bootstrap, React Router
- **Dashboard:** React.js, Chart.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas

## ✨ Features
- 🏠 Landing page with modern UI
- 📝 User Signup & Login
- 📊 Trading Dashboard
- 📈 Holdings & Positions tracking
- 📋 Orders management
- 💰 Funds overview
- 🔒 User authentication
- 👤 Profile with Sign Out

## 📦 Installation

### Clone the repo
```bash
git clone https://github.com/mdraffea/TradeSphere.git
cd TradeSphere
```

### Install dependencies
```bash
cd backend && npm install
cd ../frontend && npm install
cd ../dashboard && npm install
```

### Setup environment
Create `.env` file in backend folder:

### Run the project
```bash
# Terminal 1 - Backend
cd backend && npm start

# Terminal 2 - Frontend
cd frontend && npm start

# Terminal 3 - Dashboard
cd dashboard && npm start
```

## 🌐 Ports
- Frontend → http://localhost:3000
- Dashboard → http://localhost:3001
- Backend API → http://localhost:3002
